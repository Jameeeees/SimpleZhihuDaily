* Start on 2016.08.29

## 功能列表

* 查看文章内容(包括往日文章)
* 查看知乎讨论
* 查看专栏文章
* 查看文章评论及赞数
* 登录知乎
* 查看个人基本信息
* 无图模式
* 推送功能(施工中)
* 分享功能(施工中)


## 意见反馈
* 因为前期设计的坑，如果要浏览历史文章，请等待当前页面加载完成后再点击前一天的内容，否则会导致程序崩溃。
